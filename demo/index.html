Scroll-Incr

jQuery plugin for number increment animation when an element becomes visibile.

Follow @iam_e200

Intro

jQuery plugin for number increment animation when an element becomes visibile on the viewport while scrolling.

Usage

Download or install via npm. 

npm i scroll-incr --save

Now, in the HTML:

<!DOCTYPE html>
<html>
<head>
...
<script src="/jquery.min.js"></script>
<script src="/scroll-incr.min.js"></script>
</head>
<body>
<div class="scroll-incr" data-max="10"></div>
<div class="scroll-incr" data-max="1000" data-increment="10"></div>
<div class="scroll-incr" data-max="599" data-duration="8"></div>
<div class="scroll-incr" data-min="100" data-max="10000"></div>
</body>
</html>

Make sure to include the jQuery library before include Scroll-Incr plugin.

Example

<div class="box">
  <p>Charles Babbage designs his first mechanical computer in</p>
  <div class="scroll-incr" data-max="4">4</div>
</div>

<div class="box">
  <p>Tim Berners-Lee at CERN releases first website in</p>
  <div class="scroll-incr" data-max="1991">1991</div>
</div>

<div class="box">

  <div class="scroll-incr" data-max="1185000000">1185000000</div>
</div>

<div class="box">

</div>

</div>

<div class="box">
  <p>Nº of heart beats for one day</p>
  <div class="scroll-incr" data-max="10000" data-duration="5">10000</div>
</div>

Options

Scroll-Incr provides 4 options:

Property Description
data-min Start number for incrementing
data-max End number for incrementing
data-incr Increment by value for each rolling
data-duration Total seconds to complete the number increment
